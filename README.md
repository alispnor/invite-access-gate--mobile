# Mobile Projects

Repositorio centralizado para projetos de aplicativos mobile.

## Estrutura

Cada aplicativo fica em sua propria pasta na raiz do repositorio:

```
/
├── mobile/          # invite-access-gate
├── app-xxx/         # (futuro)
└── ...
```

## Requisitos

- Node.js >= 18
- Expo CLI
- npm ou yarn

## Como usar

```bash
# Acesse a pasta do projeto desejado
cd <nome-do-app>

# Instale as dependencias
npm install

# Inicie o projeto
npx expo start
```
